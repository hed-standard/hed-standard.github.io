Hierarchical Event Descriptors (HED)

Linking brain, body, behavior and experience in experiments through metadata to enable analysis

Why HED?

A very important question, for any neuroimaging time series data - What exactly happened during this recording?

HED annotations record and store detailed answers to this question with the data - in a form readable by both human and machines.

Who uses HED?

Experimenters

performing experiments that acquire data

Data annotators

organizing data and tagging events

Data analysts

applying HED tools to find and analyze data

Tool developers

contributing to the growing HED toolkit

Schema builders

extending HED vocabulary in new directions
